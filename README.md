# PHP Redis caching class

Redis is an open-source, in-memory data structure store that can be used as a database, cache, and message broker.

This class code allows you to store, get and delete data stored in Redis server by key name. It uses PHP Redis library via Composer.

## Requirements

- Composer.
- PHP v7 or higher.
- Redis server.
